# 🩺 SwasthAI – Mobile Anemia Detection AI

SwasthAI is an AI-powered healthcare platform that enables fast, non-invasive anemia screening using smartphone images. The system analyzes visual indicators such as conjunctival and nail-bed pallor through computer vision and machine learning models to estimate anemia risk and generate an instant health assessment.

Built with accessibility in mind, SwasthAI aims to support early detection of anemia in rural and underserved communities where access to laboratory testing and healthcare facilities may be limited. By leveraging AI, mobile technology, and edge computing, the platform offers a low-cost and scalable screening solution that can be deployed on smartphones, web applications, and edge devices such as Raspberry Pi.

Beyond anemia detection, SwasthAI serves as the foundation for a broader AI-driven healthcare ecosystem focused on improving preventive care, enabling remote health monitoring, and expanding access to intelligent healthcare services across India.

**Tech Stack:** Python, TensorFlow, OpenCV, Flask, Flutter, Docker, and Raspberry Pi.

**Mission:** To make early healthcare screening affordable, accessible, and available to everyone through AI.
